# what is AWS VPC

Amazon VPC (Virtual Private Cloud) is a **logically isolated network** inside AWS where you launch your cloud resources.

> 🏠 Your own private data center network inside AWS


# POINT TO REMEMBER

- A VPC belongs to a single AWS Region.
> One VPC = One Region = Region level
- In one region there can be multiple VPC.
- Inside that VPC you can create multiple subnets in different or in same Availability Zones(AZs).
